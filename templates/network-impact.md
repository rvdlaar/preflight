# Netwerkimpactbeoordeling / Network Impact Assessment

> **Doel / Purpose:** Structured assessment of network impact for a proposed system or integration. Covers bandwidth requirements, protocol/port specifications, network zone placement, firewall rules, QoS, wireless coverage, and capacity overlay analysis. Feeds into the PSA and is reviewed by the network architect and infrastructure team.
